name: iScroll product vision
description: Long-term product direction beyond the current MVP
type: project
---

The app is a TikTok-style snap-scroll learning feed. Current MVP has static seeded cards with chain links.

Planned future features:
1. **Adaptive depth** — track which topics/chains a user engages with (time spent, chain taps, saves) and gradually serve deeper content on those topics
2. **PDF/document ingestion** — user uploads a textbook, lecture notes, or article; app parses it and generates learn cards from it, teaching the material progressively over time
3. **Blended learning** — mix user-uploaded content cards with general curiosity cards (the random wow facts) so it never feels like pure studying

**Why:** User wants to learn actively (from their own material) while also being surprised by tangential knowledge — the "rabbit hole" feeling of the internet but structured.
